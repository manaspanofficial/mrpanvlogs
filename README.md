## 🌐 Live Site
Deployed on Vercel: [mr-pan-vlogs.vercel.app](https://mr-pan-vlogs.vercel.app)

- 🎨 **Premium Dark Theme** with orange/gold gradient accents
- 📱 **Fully Responsive** — works on mobile, tablet & desktop
- 🎬 **Hero Section** with animated orbs, floating cards & stats counter
- 👤 **About Section** with skill tags
- 📹 **Videos Gallery** (6 latest videos) with hover effects

- 📬 **Contact Form** with social links

## 🚀 Deploy on Vercel via GitHub

### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com) → **New Repository**
2. Name it `mr-pan-vlogs`
3. Keep it **Public**
4. Click **Create repository**

### Step 2: Push Files to GitHub
Run these commands in the project folder:
```bash
git init
git add .
git commit -m "🎬 Initial commit - Mr Pan Vlogs website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mr-pan-vlogs.git
git push -u origin main
```

### Step 3: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) → **Sign up with GitHub**
2. Click **"Add New Project"**
3. Select your `mr-pan-vlogs` repository
4. Click **Deploy** — done! 🎉

Your site will be live at `https://mr-pan-vlogs.vercel.app`

## 🛠️ Customization
- Update **YouTube channel link** in `index.html` (search for `@MrPanVlogs`)
- Update **social media links** in the Contact & Footer sections
- Update **email** in the contact section
- Replace stat numbers (subscribers, videos, places) with real numbers
